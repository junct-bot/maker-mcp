# Maker MCP Server

Hosted MCP server for Maker, powered by Junct.

## Connect

Add to your MCP config:

```json
{
  "mcpServers": {
    "maker": {
      "url": "https://maker.mcp.junct.dev/mcp",
      "transport": "streamable-http"
    }
  }
}
```

No authentication required. Free to use.

- **Endpoint:** https://maker.mcp.junct.dev/mcp
- **Transport:** Streamable HTTP
